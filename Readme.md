# Brand Visibility Score (LLM-Based)

This project measures how often a brand appears in AI discovery responses
using multiple LLMs (OpenAI + Gemini).

## Features
- Dual LLM visibility (OpenAI, Gemini)
- Semantic keyword expansion
- Prompt-level visibility analysis
- MongoDB storage
- Streamlit dashboard

## Tech Stack
- FastAPI
- Streamlit
- OpenAI
- Gemini
- MongoDB
